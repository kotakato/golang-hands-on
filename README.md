# golang-hands-on

## インストール

```
go get github.com/kotakato/golang-hands-on
```

## 使い方

```
Usage:

sansan-idp [OPTIONS] (users|departments|check)

Options:

  -api-key string
        API key of Sansan OpenAPI
  -endpoint string
        Endpoint of Sansan OpenAPI (default "https://api.sansan.com/v2.1")
```
